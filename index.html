üîç What Was the Error?

During the execution of market_basket() in TF-COMB, the following error appeared: 

ValueError: var_name=['TF2'] must be a scalar.

The issue occurred because newer versions of pandas changed the way pd.melt() handles parameters. Earlier, list arguments were allowed for id_vars and var_name. However, in the latest versions, they must be passed as single string values (scalars). The original TF-COMB source code used lists, leading to this error when run with modern pandas.

üß© What Was Done

The internal objects.py file in TF-COMB was updated to align with the new pandas.melt() syntax. This modification ensures TF-COMB runs correctly on current environments (tested with pandas ‚â• 2.3) and avoids compatibility issues during installation or analysis.
